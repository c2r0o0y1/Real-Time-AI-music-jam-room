# Real-Time AI Music Jam Room

## Docker Development

Run full app:

```bash
docker compose up --build
```

Frontend:

`http://localhost:5173`

Backend health:

`http://localhost:8000/health`

WebSocket:

`ws://localhost:8000/ws/session/demo-session`

Stop containers:

```bash
docker compose down
```

Rebuild:

```bash
docker compose up --build
```
